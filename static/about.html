Who built this and why

Hi, I'm Mayank

I'm Mayank Sharma, the former CTO of Kuvera. I built this MCP server as a personal side project to explore the Model Context Protocol and make it easier for people to interact with their investment portfolios using AI assistants.

This is an independent, open-source project — not affiliated with or endorsed by Kuvera. No commercial terms, no monetization, no data selling. Just a fun experiment to see what's possible when you connect AI to financial data.

Why this exists

AI assistants are becoming powerful tools for reasoning about data, but they're only as useful as the data they can access. MCP (Model Context Protocol) is a step toward standardizing that access.

Kuvera has a clean API, and I thought it would be interesting to expose it through MCP so users can ask natural-language questions about their portfolios. No more logging into a website to check your XIRR or see which stocks you're indirectly exposed to through mutual funds.

The source code is open so you can audit it, fork it, or self-host it. Transparency matters when you're dealing with financial data.
